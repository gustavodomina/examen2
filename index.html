FrontEnd

El frontend es la parte visible de un sitio web o aplicación con la que los usuarios pueden interactuar directamente. Es la cara del sitio, ubicada en el lado del cliente. El frontend se encarga de la conexión con el servidor y la base de datos utilizados por el sitio web 123. El frontend abarca menús desplegables, imágenes, colores, iconos, animaciones y una experiencia de navegación del lado frontal o delantero Las tecnologías más utilizadas en frontend son HTML, CSS y JavaScript 12. HTML es el lenguaje de marcado que define la estructura de la página web, CSS es el lenguaje de estilo que define la apariencia de la página web y JavaScript es el lenguaje de programación que permite añadir interactividad y dinamismo a la página web 1

BackEnd

El backend es la parte de un sitio web o aplicación que se encarga de la lógica y el procesamiento de datos en el lado del servidor 1. Es decir, es la parte invisible del sitio web que se encarga de la gestión de bases de datos, la autenticación de usuarios, la lógica empresarial y la comunicación con el frontend 12. El backend se encarga de recibir las solicitudes del frontend, procesarlas y enviar una respuesta al frontend 1. Las tecnologías más utilizadas en backend son JavaScript con Node.js, Python, Ruby, .NET, PHP y Java 3. Cada tecnología tiene sus propias características y limitaciones. Por ejemplo, JavaScript con Node.js permite un desarrollo rápido y minimiza los gastos generales 3. Python y Ruby son adecuados para crear aplicaciones rápidas 3. .NET es muy utilizado en el mundo empresarial y estuvo ligado a Windows en un principio 3. PHP y Java son los lenguajes de backend más utilizados en general

Tecnologias Agiles

Las metodologías ágiles son un conjunto de prácticas y valores que se utilizan en el desarrollo de software y otros proyectos para mejorar la eficiencia, la calidad y la satisfacción del cliente 12. Las metodologías ágiles se caracterizan por ser iterativas, colaborativas, adaptables y centradas en el cliente 1. Algunas de las metodologías ágiles más utilizadas son Scrum, Kanban, Extreme Programming (XP), Lean, Crystal y Feature Driven Development (FDD) 1. Las metodologías ágiles se basan en el Manifiesto Ágil, que establece cuatro valores fundamentales: individuos e interacciones, software funcionando, colaboración con el cliente y respuesta al cambio 2. Las metodologías ágiles también promueven la comunicación abierta, la retroalimentación continua, la entrega temprana y frecuente de software y la mejora continua 1
